# 🧩 数独乐园

为 6 岁左右孩子设计的网页版数独游戏。纯静态页面，零依赖、零构建、可离线玩。

## 怎么玩

直接双击打开 `index.html`，或启动本地服务器：

```bash
python3 -m http.server 8642
# 浏览器打开 http://localhost:8642
```

## 功能

- **棋盘大小自由选**：4×4 / 5×5 / 6×6 / 7×7 / 8×8 / 9×9
  - 4、6、8、9 宫格为标准数独规则（行、列、宫不重复）
  - 5、7 为拉丁方阵规则（行、列不重复），适合儿童
- **五套符号主题**：动物 🐶、水果 🍎、交通 🚗、海洋 🐳、数字（彩虹配色）
- **三档难度**：⭐ 简单 / ⭐⭐ 普通 / ⭐⭐⭐ 挑战，所有谜题保证唯一解
- **幼儿友好交互**：点格子再点图案，或先选图案当“印章”连续盖；橡皮擦、撤销、提示
- **温和纠错**：填错的格子轻轻摇晃标红，不惩罚
- **观察辅助**：同图案高亮、行列高亮、相邻宫交替底色、面板剩余计数
- **成就感**：完成有彩纸 + 欢呼音效 + 鼓励语，按难度收集星星（localStorage 存档）
- **音效**：Web Audio 合成，无外部资源，可一键静音
- 键盘支持：数字键填入、退格删除、方向键移动

## 项目结构

```
index.html          页面骨架
css/style.css       儿童友好样式
js/boxes.js         各尺寸的宫格划分
js/solver.js        回溯解算器（求解 / 解数量统计）
js/generator.js     出题器（生成完整盘 + 挖洞保唯一解）
js/state.js         游戏状态（不可变更新，支持撤销）
js/themes.js        符号主题
js/audio.js         音效
js/confetti.js      庆祝彩纸动画
js/ui.js            棋盘与符号面板渲染
js/main.js          主流程装配
tests/              单元测试
docs/superpowers/specs/  设计文档
```

## 测试

```bash
node --test tests/*.test.js
```
